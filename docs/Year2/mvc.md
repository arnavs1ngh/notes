# Multi-Variable Calculus & Differential Equations

$$ \begin{aligned}
Z &= \frac{Pv}{RT} = 1 + \frac{B}{v} + \frac{C}{v^2} + ... \\
B &= \sum_i \sum_j x_ix_jB_{ij}(T)\\
C &= \sum_i \sum_j \sum_k x_ix_jx_kC_{ijk}(T), etc.
\end{aligned} $$
